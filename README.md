# Hangarin

### Project Description
**Hangarin** Hangarin is a simple yet powerful web application built with Django that helps users organize their daily tasks, manage priorities, categorize work, and stay productive.

### Features 
* **Task Management**: Create, update, and delete tasks Set deadlines and descriptions and Track task status: Pending, In Progress, Completed

* **Relational Database Modeling**: Developed five core models (*College, Program, Organization, Student,* and *OrgMember*) with automated timestamps and relational integrity.
* **Dashboard**: Overview of:

    * **Total tasks**
    * **Pending tasks**
    * **Tasks in progress**
    * **Completed tasks**
    * **Displays recent tasks**
